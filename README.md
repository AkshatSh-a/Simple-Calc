# Simple Calculator

This is a simple calculator web application built using HTML, CSS, and JavaScript. The calculator can perform basic arithmetic operations like addition, subtraction, multiplication, and division. It features a dark mode design with circular buttons and includes special styling for the "Clear" and "Equals" buttons.

## Features

- **Dark Mode**: The calculator has a sleek, dark-themed interface for a modern look.
- **Circular Buttons**: All buttons have a circular shape, creating a stylish 3D effect with shadows.
- **Basic Operations**: Supports addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`).
- **Clear and Equals Buttons**: The "C" (Clear) and "=" (Equals) buttons are highlighted with an orange background for better visibility.


## How to Use

1. Clone or download the repository.
2. Open the `index.html` file in your web browser.
3. Use the calculator to perform basic arithmetic operations.
